# gotools

## HTTP server

### Install

```sh
go install github.com/o7q2ab/gotools/cmd/gosrv@latest
```

### Use

```sh
gosrv 8888
```

## Go processes searcher

### Install

```sh
go install github.com/o7q2ab/gotools/cmd/goproc@latest
```

### Use

```sh
goproc
```

## Go binaries info

### Install

```sh
go install github.com/o7q2ab/gotools/cmd/goxm@latest
```

### Use

```sh
goxm ~/go/bin/gopls
```
